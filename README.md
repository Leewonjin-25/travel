## 🛠️ 제공 서비스 (Service Features)

### 1️⃣ 스마트 장소 검색
- 전국의 문화시설 데이터를 **지역별(시도/시군구)**로 분류
- **카테고리(전시, 공연, 레저 등)** 기반 맞춤형 결과 노출

### 2️⃣ 배리어프리 체크리스트
- **이동 편의**: 장애인 전용 주차장, 전용 출입문, 휠체어 대여
- **감각 보조**: 시각장애인 안내견 동반, 점자 가이드 제공
- **기본 정보**: 운영 시간, 무료/유료 주차 및 입장료 정보

### 3️⃣ 20km 답사 코스 메이커
- 선택한 장소를 기점으로 **반경 20km 이내**의 장소를 선별
- "너무 멀지 않은" 이동 가능한 최적의 코스 3개를 생성

## 🧭 이용 안내 (How to Use)

| 단계 | 활동 | 내용 |
| :--- | :--- | :--- |
| **STEP 1** | **지역 선택** | 거주 중인 시/도와 시/군/구를 선택하세요. |
| **STEP 2** | **카테고리 설정** | 보고 싶은 문화예술 장소의 종류를 고르세요. |
| **STEP 3** | **코스 생성** | 추천 코스 버튼을 눌러 이동 경로를 확인하세요. |
| **STEP 4** | **상세 확인** | 하단 리스트에서 편의시설(화장실, 주차 등) 정보를 체크하세요! |
